# 视频通话 — 设计思路与实现

## 一、设计思路

### 1.1 核心需求

HR 和求职者之间进行实时视频面试：
- HR 发起邀约 → 求职者弹窗确认 → 双方视频通话
- 支持 WebRTC P2P 直连（低延迟）
- P2P 失败时自动回退到 WebSocket 中继（Relay）
- 通话结束记录时长

### 1.2 方案选型

| 方案 | 优点 | 缺点 | 结论 |
|------|------|------|------|
| WebRTC P2P | 延迟低、不占服务器带宽 | 对称 NAT 穿透困难 | **首选** |
| WebSocket Relay | 穿透所有 NAT | 占服务器带宽、有延迟 | **P2P 失败时回退** |
| SFU/MCU | 专业视频方案 | 需要额外部署 Kurento/Jitsi | 不采用 |

**为什么需要 Relay 回退？**

测试发现两台 PC 在不同网络环境下都是对称 NAT（Symmetric NAT），STUN 映射的公网地址互不可达，OpenRelay 免费 TURN 服务器全部超时。P2P 连接必然失败，因此必须有服务端中继作为保底。

### 1.3 信令复用聊天 WebSocket

所有视频信令（invite/accept/reject/offer/answer/ice/hangup）通过已有的 `ws/chat` WebSocket 传输，不需要额外的信令服务器：

```
信令消息: JSON 文本帧（复用聊天 WS）
媒体数据: 二进制帧（Relay 模式下通过聊天 WS 传输）
```

### 1.4 Relay 模式的两种编码策略

| 策略 | 编码器 | 解码器 | 兼容性 |
|------|--------|--------|--------|
| **WebCodecs** | `VideoEncoder` (VP8) | `VideoDecoder` → Canvas 绘制 | Chrome/Edge 94+ |
| **MediaRecorder** | `MediaRecorder` (webm/opus) | `MediaSource` + `SourceBuffer` | 所有现代浏览器 |

优先使用 WebCodecs（性能更好），不支持时降级到 MediaRecorder。

### 1.5 二进制帧协议（无 base64 开销）

```
字节偏移    长度       说明
─────────────────────────────────────
0           1 字节     标志位（bit0 = init 段）
1-2         2 字节     codec 字符串长度（大端序）
3-N         N 字节     codec 字符串（如 "vp8" 或 "video/webm; codecs=vp8,opus"）
N+1-N+8     8 字节     发送者 userId（大端序 long，拆为两个 32 位写入）
N+9+        剩余       编码后的视频/音频数据
```

```java
// ChatWebSocket.java — 二进制帧处理（视频中继数据）
@Override
protected void handleBinaryMessage(WebSocketSession session, BinaryMessage message) {
    SessionInfo info = sessionInfoMap.get(session.getId());
    if (info == null) return;

    java.nio.ByteBuffer buffer = message.getPayload();
    int totalSize = buffer.remaining();

    // 最小帧：1B flags + 2B codecLen + 8B senderId = 11B（仅做边界检查，不解析）
    if (totalSize < 11) {
        log.warn("二进制帧过小: {}B, userId={}", totalSize, info.userId);
        return;
    }

    // 零解析转发：不读取头部、不反序列化视频数据
    Long receiverId = parseReceiverId(info.conversationId, info.userId);
    sendBinaryToUser(receiverId, info.conversationId, buffer.duplicate());

    // 采样日志：每 200 帧打印一次
    long count = ++relayFrameCount;
    if (count <= 2 || count % 200 == 0) {
        log.info("视频二进制中继: from={} to={}, size={}B, frame#{}", info.userId, receiverId, totalSize, count);
    }
}

// 二进制帧定向发送
public void sendBinaryToUser(Long userId, String conversationId, java.nio.ByteBuffer data) {
    Set<WebSocketSession> sessions = conversationSessions.get(conversationId);
    if (sessions == null) return;
    for (WebSocketSession session : sessions) {
        SessionInfo info = sessionInfoMap.get(session.getId());
        if (info != null && info.userId().equals(userId) && session.isOpen()) {
            data.rewind(); // 多 session 时确保 position 正确
            session.sendMessage(new BinaryMessage(data));
        }
    }
}
```

```javascript
]

// 创建 PeerConnection 并绑定事件
export function createPeerConnection({ onRemoteStream, onIceCandidate, onConnected, onDisconnected, onError }) {
  const pc = new RTCPeerConnection({
    iceServers: ICE_SERVERS,
    iceCandidatePoolSize: 2 // 预分配候选，加速 ICE 收敛
  })

  pc.onicecandidate = (event) => {
    if (event.candidate) onIceCandidate?.(event.candidate)
  }

  pc.ontrack = (event) => {
    if (event.streams?.[0]) onRemoteStream?.(event.streams[0])
  }

  pc.oniceconnectionstatechange = () => {
    const state = pc.iceConnectionState
    if (state === 'connected' || state === 'completed') onConnected?.()
    else if (state === 'failed' || state === 'disconnected') onDisconnected?.()
  }

  return pc
}
```

```javascript
// webrtc.js — 获取本地摄像头+麦克风
export async function getLocalStream() {
  try {
    // 优先：视频+音频
    return await navigator.mediaDevices.getUserMedia({
      video: {
        width: { ideal: 1280 }, height: { ideal: 720 },
        frameRate: { ideal: 30, max: 30 }, facingMode: 'user'
      },
      audio: { echoCancellation: true, noiseSuppression: true, autoGainControl: true }
    })
  } catch {
    // 降级：只获取摄像头，再尝试单独添加音频
    const stream = await navigator.mediaDevices.getUserMedia({
      video: { width: { ideal: 1280 }, height: { ideal: 720 }, frameRate: { ideal: 30, max: 30 } },
      audio: false
    })
    try {
      const audioStream = await navigator.mediaDevices.getUserMedia({ audio: true })
      audioStream.getAudioTracks().forEach(track => stream.addTrack(track))
    } catch { /* 无麦克风也行 */ }
    return stream
  }
}
```

```javascript
// MessageCenter.vue — P2P 连接管理
let iceRestartCount = 0
const MAX_ICE_RESTART = 2
let connectTimeoutTimer = null

const initPeerConnection = async (isInitiator) => {
  if (peerConnection) return
  iceRestartCount = 0

  peerConnection = createPeerConnection({
    onRemoteStream: (stream) => {
      // 已切换到 Relay 模式时，忽略迟到的 P2P 流
      if (relayMode.value) return
      remoteStream.value = stream
      videoCallStatus.value = ''
    },
    onIceCandidate: (candidate) => {
      wsSend({ type: 'video_ice', candidate: candidate.toJSON() })
    },
    onConnected: () => {
      // P2P 连接成功，取消超时定时器
      if (connectTimeoutTimer) { clearTimeout(connectTimeoutTimer); connectTimeoutTimer = null }
      videoCallStatus.value = ''
      iceRestartCount = 0
    },
    onDisconnected: () => {
      if (peerConnection && iceRestartCount < MAX_ICE_RESTART) {
        // 尝试 ICE 重启
        iceRestartCount++
        videoCallStatus.value = `连接中断，正在尝试恢复 (${iceRestartCount}/${MAX_ICE_RESTART})...`
        restartIce(peerConnection).then(offer => {
          wsSend({ type: 'video_offer', sdp: offer })
        }).catch(() => switchToRelayMode())
      } else {
        // 重启耗尽 → 回退 Relay
        switchToRelayMode()
      }
    }
  })

  // 3 秒超时：P2P 未连上就回退
  connectTimeoutTimer = setTimeout(() => {
    connectTimeoutTimer = null
    if (peerConnection && peerConnection.iceConnectionState !== 'connected'
        && peerConnection.iceConnectionState !== 'completed') {
      switchToRelayMode()
    }
  }, 3000)

  // 发起方：创建 Offer
  if (isInitiator) {
    await startLocalVideo()
    videoCallStatus.value = '正在建立连接...'
    videoCallVisible.value = true
    const offer = await createOffer(peerConnection, localStream.value)
    wsSend({ type: 'video_offer', sdp: offer })
  }
}
```

```javascript
// webrtc.js — 启动 WebSocket Media Relay
export function startMediaRelay(localStream, sendFn, options = {}) {
  if (relayMediaRecorder) return
  const { sendBinary, senderId } = options

  // 选择编码格式（VP8 优先）
  const mimeType = MediaRecorder.isTypeSupported('video/webm; codecs=vp8,opus')
    ? 'video/webm; codecs=vp8,opus'
    : MediaRecorder.isTypeSupported('video/webm; codecs=vp9,opus')
      ? 'video/webm; codecs=vp9,opus'
      : 'video/webm'

  // 限制画质
  const videoTrack = localStream.getVideoTracks()[0]
  if (videoTrack) {
    videoTrack.applyConstraints({
      width: { ideal: 1280 }, height: { ideal: 720 },
      frameRate: { ideal: 15, max: 18 }
    }).catch(() => {})
  }

  relayMediaRecorder = new MediaRecorder(localStream, {
    mimeType,
    videoBitsPerSecond: 800000,  // 800kbps 视频
    audioBitsPerSecond: 128000   // 128kbps 音频
  })

  let relaySendCount = 0

  relayMediaRecorder.ondataavailable = (e) => {
    if (e.data.size === 0) return
    relaySendCount++
    const isInit = relaySendCount === 1

    e.data.arrayBuffer().then(buffer => {
      if (sendBinary && senderId) {
        // 二进制帧模式：无 base64 开销
        const frame = buildBinaryFrame(buffer, isInit, mimeType, senderId)
        sendBinary(frame)
      } else {
        // 降级：base64 + JSON 文本帧
        const bytes = new Uint8Array(buffer)
        let binary = ''
        for (let i = 0; i < bytes.length; i++) binary += String.fromCharCode(bytes[i])
        sendFn({ type: 'video_media', data: btoa(binary), init: isInit, codec: mimeType })
      }
    })
  }

  relayMediaRecorder.start(180) // 每 180ms 切割一次
}

// 停止 Relay
export function stopMediaRelay() {
  if (relayMediaRecorder && relayMediaRecorder.state !== 'inactive') {
    relayMediaRecorder.stop()
  }
  relayMediaRecorder = null
  // 同时停止 WebCodecs 编码器
  if (webCodecsEncoder) { try { webCodecsEncoder.close() } catch {}; webCodecsEncoder = null }
  if (webCodecsRafId) { cancelAnimationFrame(webCodecsRafId); webCodecsRafId = null }
  if (webCodecsAudioRecorder && webCodecsAudioRecorder.state !== 'inactive') {
    try { webCodecsAudioRecorder.stop() } catch {}
  }
  webCodecsAudioRecorder = null
}
```

### 2.10 前端：WebCodecs 编码发送（webrtc.js）

```javascript
// webrtc.js — WebCodecs 视频编码
export function startWebCodecsSender(localStream, sendBinaryFn, senderId, options = {}) {
  if (webCodecsEncoder) return

  const videoTrack = localStream.getVideoTracks()[0]
  if (!videoTrack) return

  const settings = videoTrack.getSettings()
  const nativeWidth = Math.min(settings.width || 1280, 1920)
  const nativeHeight = Math.min(settings.height || 720, 1080)
  const nativeFps = Math.min(settings.frameRate || 30, 30)

  const bitrate = 2500000 // 2.5Mbps
  const frameInterval = 1000 / nativeFps
  let sendCount = 0
  let lastFrameTime = 0

  // 创建 VP8 编码器
  const encoder = new VideoEncoder({
    output: (chunk, metadata) => {
      const buffer = new ArrayBuffer(chunk.byteLength)
      chunk.copyTo(buffer)
      sendCount++
      const frame = buildBinaryFrame(buffer, chunk.type === 'key', 'vp8', senderId)
      sendBinaryFn(frame)
    },
    error: (e) => { webCodecsEncoder = null }
  })

  encoder.configure({
    codec: 'vp8', width: nativeWidth, height: nativeHeight,
    bitrate, framerate: nativeFps, latencyMode: 'realtime'
  })

  webCodecsEncoder = encoder

  // MediaStreamTrackProcessor 采集帧
  const processor = new MediaStreamTrackProcessor({ track: videoTrack })
  const reader = processor.readable.getReader()

  const processFrames = async () => {
    while (webCodecsEncoder === encoder) {
      const { value: frame, done } = await reader.read()
      if (done || webCodecsEncoder !== encoder) { if (frame) frame.close(); break }

      const now = performance.now()
      // 按帧率限流
      if (now - lastFrameTime < frameInterval) { frame.close(); continue }
      lastFrameTime = now

      // 编码队列过长时跳帧
      if (encoder.encodeQueueSize > 2) { frame.close(); continue }

      const makeKeyFrame = sendCount === 0 || sendCount % 72 === 0 // 每 3 秒一个关键帧
      encoder.encode(frame, { keyFrame: makeKeyFrame })
      frame.close()
    }
    reader.releaseLock()
  }

  processFrames().catch(e => {
    if (webCodecsEncoder === encoder) { webCodecsEncoder = null; encoder.close() }
    if (options.onError) options.onError() // 降级到 MediaRecorder
  })

  // 音频：用独立 MediaRecorder 只录音频
  const audioTrack = localStream.getAudioTracks()[0]
  if (audioTrack) {
    const audioStream = new MediaStream([audioTrack])
    let audioMimeType = 'audio/webm; codecs=opus'
    let audioCodec = 'opus-audio'
    if (MediaRecorder.isTypeSupported('audio/mp4; codecs=mp4a.40.2')) {
      audioMimeType = 'audio/mp4; codecs=mp4a.40.2'
      audioCodec = 'aac-audio'
    }

    const audioRecorder = new MediaRecorder(audioStream, { mimeType: audioMimeType, audioBitsPerSecond: 128000 })
    let audioSendCount = 0
    audioRecorder.ondataavailable = (e) => {
      if (e.data.size === 0) return
      audioSendCount++
      e.data.arrayBuffer().then(buffer => {
        const frame = buildBinaryFrame(buffer, audioSendCount === 1, audioCodec, senderId)
        sendBinaryFn(frame)
      })
    }
    audioRecorder.start(180)
    webCodecsAudioRecorder = audioRecorder
  }
}
```

### 2.11 前端：二进制帧构建（webrtc.js）

```javascript
// webrtc.js — 构建二进制帧
function buildBinaryFrame(videoBuffer, isInit, codec, senderId) {
  const codecBytes = new TextEncoder().encode(codec)
  const headerLen = 1 + 2 + codecBytes.length + 8
  const frame = new ArrayBuffer(headerLen + videoBuffer.byteLength)
  const view = new DataView(frame)
  const u8 = new Uint8Array(frame)

  // 1B flags：bit0 = init
  view.setUint8(0, isInit ? 1 : 0)
  // 2B codec 长度（big-endian）
  view.setUint16(1, codecBytes.length, false)
  // NB codec 字符串
  u8.set(codecBytes, 3)
  // 8B senderId（big-endian，分两次写入规避 JS 精度限制）
  view.setUint32(3 + codecBytes.length, Math.floor(senderId / 0x100000000), false)
  view.setUint32(3 + codecBytes.length + 4, senderId >>> 0, false)
  // 剩余：原始视频数据
  u8.set(new Uint8Array(videoBuffer), headerLen)

  return frame
}
```

### 2.12 前端：接收方解析二进制帧（MessageCenter.vue）

```javascript
// MessageCenter.vue — 处理收到的二进制视频帧
const handleBinaryVideoFrame = (buffer) => {
  const view = new DataView(buffer)
  if (buffer.byteLength < 11) return // 最小帧大小

  const flags = view.getUint8(0)
  const isInit = (flags & 1) === 1
  const codecLen = view.getUint16(1, false) // big-endian
  const codecBytes = new Uint8Array(buffer, 3, codecLen)
  const codec = new TextDecoder().decode(codecBytes)
  const senderIdHigh = view.getUint32(3 + codecLen, false)
  const senderIdLow = view.getUint32(3 + codecLen + 4, false)
  const senderId = senderIdHigh * 0x100000000 + senderIdLow
  const videoData = new Uint8Array(buffer, 3 + codecLen + 8)

  // 忽略自己的帧
  if (senderId === currentUserId.value || videoCallEnded) return

  const relayData = { init: isInit, codec, data: videoData }

  if (videoInviteAccepted) {
    // 音频帧路由到独立的 audioReceiver
    if (codec === 'opus-audio' || codec === 'aac-audio' || codec === 'webm-audio') {
      if (!audioReceiver) audioReceiver = createAudioReceiver()
      audioReceiver.handleData(relayData)
      return
    }

    // 视频帧路由到 relayReceiver（WebCodecs 或 MediaRecorder）
    const rawRef = videoCallRef.value?.remoteVideoEl
    const videoEl = rawRef?.value ?? rawRef
    if (videoEl && videoEl instanceof HTMLElement) {
      if (!relayReceiver) {
        // 根据 codec 选择正确的接收器
        if (codec === 'vp8' && isWebCodecsSupported()) {
          relayReceiver = createWebCodecsReceiver()
        } else if (codec?.startsWith('video/webm') || !isWebCodecsSupported()) {
          relayReceiver = createRelayReceiver()
        } else {
          relayReceiver = isWebCodecsSupported() ? createWebCodecsReceiver() : createRelayReceiver()
        }
      }
      relayReceiver.handleData(relayData, videoEl)
      return
    }
  }

  // videoEl 未就绪，缓存到 buffer
  relayDataBuffer.push(relayData)
  startRelayDrainPolling()
}
```

### 2.13 前端：MediaSource 播放接收器（webrtc.js）

```javascript
// webrtc.js — MediaSource 接收器（MediaRecorder 模式）
export function createRelayReceiver() {
  let mediaSource = null
  let sourceBuffer = null
  let videoEl = null
  let queue = []
  let initReceived = false
  let generation = 0
  let isPlaying = false

  function handleData(data, el) {
    const raw = data.data instanceof Uint8Array
      ? data.data
      : Uint8Array.from(atob(data.data), c => c.charCodeAt(0))

    if (data.init) {
      // 初始化段：创建 MediaSource + SourceBuffer
      initReceived = true
      queue = []
      videoEl = el
      generation++

      if (mediaSource) {
        try { URL.revokeObjectURL(el.src) } catch {}
        try { mediaSource.endOfStream() } catch {}
        mediaSource = null; sourceBuffer = null
      }

      mediaSource = new MediaSource()
      el.srcObject = null
      el.src = URL.createObjectURL(mediaSource)
      isPlaying = false

      const onSourceOpen = () => {
        let codec = data.codec || 'video/webm'
        // 选择支持的 codec
        if (!MediaSource.isTypeSupported(codec)) {
          codec = ['video/webm; codecs=vp8,opus', 'video/webm; codecs=vp9,opus', 'video/webm']
            .find(c => MediaSource.isTypeSupported(c)) || 'video/webm'
        }

        sourceBuffer = mediaSource.addSourceBuffer(codec)
        sourceBuffer.mode = 'sequence'
        sourceBuffer.addEventListener('updateend', drainQueue)
        sourceBuffer.appendBuffer(raw)

        if (!isPlaying) {
          el.play().then(() => { isPlaying = true }).catch(() => {})
        }
      }

      mediaSource.addEventListener('sourceopen', onSourceOpen)
      if (mediaSource.readyState === 'open') onSourceOpen()
    } else if (sourceBuffer && !sourceBuffer.updating) {
      try { sourceBuffer.appendBuffer(raw) } catch { queue.push(raw) }
    } else {
      queue.push(raw)
      if (queue.length > 200) queue.splice(0, queue.length - 100) // 防队列溢出
    }
  }

  function drainQueue() {
    if (!sourceBuffer || sourceBuffer.updating || queue.length === 0) return
    sourceBuffer.appendBuffer(queue.shift())
  }

  function cleanup() {
    generation++
    if (mediaSource) { try { mediaSource.endOfStream() } catch {}; mediaSource = null }
    sourceBuffer = null; queue = []; initReceived = false
    if (videoEl) { try { videoEl.pause(); videoEl.src = '' } catch {}; videoEl = null }
  }

  return { handleData, cleanup }
}
```

```javascript
// webrtc.js — WebCodecs 视频接收器（Canvas 渲染）
export function createWebCodecsReceiver() {
  let decoder = null
  let canvas = null
  let ctx = null
  let frameCount = 0
  let failed = false

  function handleData(data, el) {
    if (failed) return

    const raw = data.data instanceof Uint8Array
      ? data.data
      : Uint8Array.from(atob(data.data), c => c.charCodeAt(0))

    if (!decoder) {
      // 创建 Canvas 替代 video 元素
      canvas = document.createElement('canvas')
      canvas.style.cssText = 'width:100%;height:100%;object-fit:contain;background:#000'
      ctx = canvas.getContext('2d')
      el.style.display = 'none'
      el.parentElement?.appendChild(canvas)

      decoder = new VideoDecoder({
        output: (videoFrame) => {
          if (failed) { videoFrame.close(); return }
          if (canvas.width !== videoFrame.displayWidth || canvas.height !== videoFrame.displayHeight) {
            canvas.width = videoFrame.displayWidth
            canvas.height = videoFrame.displayHeight
          }
          ctx.drawImage(videoFrame, 0, 0)
          videoFrame.close()
          frameCount++
        },
        error: (e) => { failed = true; cleanup() }
      })

      decoder.configure({ codec: data.codec === 'vp8' ? 'vp8' : 'vp9' })
    }

    const chunk = new EncodedVideoChunk({
      type: data.init ? 'key' : 'delta',
      timestamp: performance.now() * 1000,
      data: raw
    })
    decoder.decode(chunk)
  }

  function cleanup() {
    if (decoder) { try { decoder.close() } catch {}; decoder = null }
    if (canvas?.parentElement) canvas.parentElement.removeChild(canvas)
    canvas = null; ctx = null; frameCount = 0
  }

  return { handleData, cleanup, getCanvas: () => canvas, get failed() { return failed } }
}
```

### 2.15 前端：音频接收器（webrtc.js）

```javascript
// webrtc.js — 独立音频接收器（双 Audio 元素交替播放）
export function createAudioReceiver() {
  let audioInitData = null
  let audioMime = 'audio/mp4'
  let audioElA = null, audioElB = null
  let audioToggle = false

  function handleData(data) {
    if (data.codec !== 'opus-audio' && data.codec !== 'aac-audio' && data.codec !== 'webm-audio') return

    const raw = data.data instanceof Uint8Array
      ? data.data
      : Uint8Array.from(atob(data.data), c => c.charCodeAt(0))

    if (data.init) {
      audioInitData = new Uint8Array(raw)
      audioMime = data.codec === 'aac-audio' ? 'audio/mp4' : 'audio/webm'
      if (!audioElA) { audioElA = document.createElement('audio'); audioElA.style.display = 'none'; audioElA.volume = 1.0; document.body.appendChild(audioElA) }
      if (!audioElB) { audioElB = document.createElement('audio'); audioElB.style.display = 'none'; audioElB.volume = 1.0; document.body.appendChild(audioElB) }
      return
    }

    if (!audioInitData) return

    // 交替选择 Audio 元素（减少 play() 频率限制）
    const el = audioToggle ? audioElB : audioElA
    audioToggle = !audioToggle

    // 合并 init + chunk → blob → 播放
    const combined = new Uint8Array(audioInitData.byteLength + raw.byteLength)
    combined.set(audioInitData, 0)
    combined.set(raw, audioInitData.byteLength)
    const blob = new Blob([combined], { type: audioMime })
    const url = URL.createObjectURL(blob)

    if (el._currentUrl) try { URL.revokeObjectURL(el._currentUrl) } catch {}
    el._currentUrl = url
    el.src = url
    el.play().catch(() => { setTimeout(() => { try { el.play() } catch {} }, 200) })
  }

  function cleanup() {
    audioInitData = null
    ;[audioElA, audioElB].forEach(el => {
      if (!el) return
      try { el.pause(); el.src = '' } catch {}
      if (el._currentUrl) try { URL.revokeObjectURL(el._currentUrl) } catch {}
      el.parentElement?.removeChild(el)
    })
    audioElA = null; audioElB = null
  }

  return { handleData, cleanup }
}
```

| ICE 重启次数 | 最多 2 次 |
| P2P 超时 | 3 秒 |

## 五、关键设计决策

| 决策 | 原因 |
|------|------|
| 信令复用聊天 WS | 不需要额外连接，减少复杂度 |
| 广播 invite/accept/reject | 会话中所有人都需要知道通话状态变化 |
| 定向 offer/answer/ice | WebRTC 信令只在双方之间传输 |
| P2P → Relay 自动回退 | 对称 NAT 环境下 P2P 必然失败，Relay 保障可用性 |
| WebCodecs 优先 | 性能优于 MediaRecorder，CPU 占用更低 |
| 二进制帧协议 | 比 base64 JSON 节省 33% 带宽，后端零解析转发 |
| 3 秒 P2P 超时 | 快速检测 P2P 失败，减少用户等待 |
| 铃声在用户手势中播放 | 浏览器自动播放策略要求用户交互后才能播放音频 |
| 音视频分离编码（WebCodecs 模式） | 音频用 MediaRecorder、视频用 WebCodecs，解耦后可独立路由 |
| Canvas 绘制（WebCodecs 接收） | VideoDecoder 输出 VideoFrame → Canvas drawImage，绕过 MediaSource 限制 |
| 双 Audio 元素交替播放 | 减少 play() 频率限制，避免音频卡顿 |
| 自定义视频邀约弹窗 | iOS Safari 要求 getUserMedia 在同步点击上下文中调用 |
| 音频预热（静音 WAV） | 维持全局音频会话解锁，供后续铃声和视频音频复用 |
| relayDataBuffer 缓冲 | videoEl 未就绪时缓存数据，就绪后立即清空 |

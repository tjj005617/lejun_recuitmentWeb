# AI 模拟面试 — 八股栏目 & 混合模式 设计文档

## 1. 需求概述

在原有「简历分析」面试模式基础上，新增两种面试模式：

| 模式 | 英文标识 | 说明 |
|------|---------|------|
| 简历分析 | `resume` | 根据用户上传的简历生成面试题（原有功能） |
| 八股栏目 | `topic` | 根据用户选择的知识图谱分类，由 AI 生成针对性八股面试题 |
| 混合模式 | `hybrid` | 同时参考简历 + 八股分类，综合出题 |

八股栏目下支持两种题型：

| 题型 | 标识 | 题量 | 交互方式 |
|------|------|------|---------|
| 简答题 | `essay` | 10 道 | 逐轮 chat 问答（文字/语音） |
| 选择题 | `choice` | 20 道 | 答题卡 UI，自动判分 |

## 5. 后端设计

### 5.1 创建面试接口（InterviewController）

```java
@PostMapping("/create")
public Result<?> createInterview(@RequestBody Map<String, Object> request) {
    // 1. 解析参数
    String interviewMode = (String) request.get("interviewMode");  // resume/topic/hybrid
    String questionType  = (String) request.get("questionType");   // essay/choice

    // 2. 分类 ID → 分类名称（用于 AI prompt）
    List<String> categoryNames = null;
    if (request.get("categoryIds") != null) {
        List<?> rawIds = (List<?>) request.get("categoryIds");
        categoryNames = new ArrayList<>();
        for (Object idObj : rawIds) {
            KgCategory cat = kgCategoryService.getById(Long.valueOf(idObj.toString()));
            if (cat != null) categoryNames.add(cat.getName());
        }
    }

    // 3. 创建面试记录
    Interview interview = interviewService.createInterview(
        userId, resumeId, jobType, interviewMode, categoryIds);
    interview.setQuestionType(questionType);
    interviewMapper.updateById(interview);

    // 4. 根据题型分支生成问题
    Resume resume = resumeId != null ? resumeService.getResumeById(resumeId) : null;
    String conversationId = interview.getId() + "_" + userId;

    List<String> questions;
    if ("choice".equals(questionType)) {
        // 选择题：20道四选一
        questions = interviewService.generateChoiceQuestions(
            interview, resume, conversationId, interviewMode, categoryNames);
    } else {
        // 简答题：10道
        questions = interviewService.generateAllQuestions(
            interview, resume, conversationId, interviewMode, categoryNames);
    }

    return Result.ok(Map.of("interview", interview, "questions", questions));
}
```

```java
// AIServiceImpl.java
@Override
public List<String> generateAllQuestions(Resume resume, String jobType,
        String conversationId, String interviewMode, List<String> categoryNames) {

    StringBuilder contextBuilder = new StringBuilder();

    // 简历上下文（resume / hybrid 模式）
    if (resume != null) {
        contextBuilder.append("候选人简历内容如下：\n")
                      .append(resume.getRawContent()).append("\n\n");
    }

    // 八股分类上下文（topic / hybrid 模式）
    if (categoryNames != null && !categoryNames.isEmpty()) {
        contextBuilder.append("请重点考察以下技术领域的八股知识：")
                      .append(String.join("、", categoryNames)).append("\n\n");
    }

    String userPrompt = String.format(
        "%s" +
        "请根据以上信息生成10道面试题，要求：\n" +
        "1. 第1题为自我介绍（如果是八股模式，第1题改为该领域的基础概念题）\n" +
        "2. 覆盖技术深度、基础知识、编码能力、系统设计\n" +
        "3. 难度从易到难递进\n" +
        // ... 更多约束
        "返回纯JSON数组，格式：\n" +
        "[{\"index\":0,\"question\":\"题目内容\",\"category\":\"分类\"}]",
        contextBuilder   // ← 动态上下文作为 %s 注入
    );

    String response = aiClient.call(systemPrompt, userPrompt, 2048, conversationId);
    return parseQuestionArray(response);  // JSON 解析 + 正则兜底
}
```

### 5.3 问题存储（InterviewServiceImpl）

```java
// 简答题：直接存文本
@Override
public List<String> generateAllQuestions(Interview interview, Resume resume,
        String conversationId, String interviewMode, List<String> categoryNames) {

    List<String> questions = aiService.generateAllQuestions(
        resume, interview.getJobType(), conversationId, interviewMode, categoryNames);

    for (int i = 0; i < questions.size(); i++) {
        InterviewQA qa = new InterviewQA();
        qa.setInterviewId(interview.getId());
        qa.setRound(i + 1);
        qa.setQuestion(questions.get(i));  // 纯文本
        interviewQAMapper.insert(qa);
    }
    return questions;
}

// 选择题：存完整 JSON（含选项、正确答案、解析）
@Override
public List<String> generateChoiceQuestions(Interview interview, Resume resume,
        String conversationId, String interviewMode, List<String> categoryNames) {

    List<Map<String, Object>> choiceQuestions = aiService.generateChoiceQuestions(
        resume, interview.getJobType(), conversationId, interviewMode, categoryNames);

    List<String> questionStrs = new ArrayList<>();
    for (int i = 0; i < choiceQuestions.size(); i++) {
        String questionJson = objectMapper.writeValueAsString(choiceQuestions.get(i));

        InterviewQA qa = new InterviewQA();
        qa.setInterviewId(interview.getId());
        qa.setRound(i + 1);
        qa.setQuestion(questionJson);  // 完整 JSON
        interviewQAMapper.insert(qa);

        questionStrs.add(questionJson);
    }

    interview.setTotalRounds(choiceQuestions.size());
    interviewMapper.updateById(interview);
    return questionStrs;
}
```

### 5.4 选择题判分（InterviewServiceImpl）

```java
@Override
public Map<String, Object> submitChoiceAnswers(Long interviewId, Map<String, String> answers) {
    List<InterviewQA> qaList = getQAHistory(interviewId);
    int correctCount = 0;
    List<Map<String, Object>> details = new ArrayList<>();

    for (InterviewQA qa : qaList) {
        int round = qa.getRound();
        String userAnswer = answers.getOrDefault(String.valueOf(round), "");

        // 从 question JSON 中提取正确答案
        Map<String, Object> qData = objectMapper.readValue(qa.getQuestion(), Map.class);
        String correctAnswer = (String) qData.get("correctAnswer");
        String explanation   = (String) qData.getOrDefault("explanation", "");

        boolean isCorrect = userAnswer.equalsIgnoreCase(correctAnswer);
        if (isCorrect) correctCount++;

        // 更新 QA 记录
        qa.setAnswer(userAnswer);
        qa.setFeedback(isCorrect ? "回答正确" : "回答错误，正确答案是 " + correctAnswer);
        qa.setReferenceAnswer(explanation);
        interviewQAMapper.updateById(qa);

        details.add(Map.of(
            "round", round,
            "userAnswer", userAnswer,
            "correctAnswer", correctAnswer,
            "correct", isCorrect,
            "explanation", explanation
        ));
    }

    // 满分 100，每题 5 分
    double totalScore = correctCount * 100.0 / qaList.size();
    Interview interview = interviewMapper.selectById(interviewId);
    interview.setTotalScore(totalScore);
    interview.setStatus("COMPLETED");
    interview.setCompletedAt(LocalDateTime.now());
    interviewMapper.updateById(interview);

    return Map.of(
        "totalScore", totalScore,
        "correctCount", correctCount,
        "total", qaList.size(),
        "details", details
    );
}
```

## 7. AI Prompt 策略

### 7.1 三种模式的 Prompt 上下文对比

| 模式 | resume 上下文 | categoryNames 上下文 | prompt 效果 |
|------|:---:|:---:|---|
| `resume` | ✅ | ❌ | 根据简历经历出题 |
| `topic` | ❌ | ✅ | 根据八股分类出题 |
| `hybrid` | ✅ | ✅ | 简历 + 八股综合出题 |

### 7.2 简答题 Prompt 结构

```
[System] 你是一位专业的技术面试官。请严格返回合法的JSON数组。

[User]
{上下文: 简历 / 分类 / 两者}

请根据以上信息生成10道面试题，要求：
1. 第1题为自我介绍（如果是八股模式，第1题改为该领域的基础概念题）

返回纯JSON数组，格式：
[{"index":0,"question":"题目内容","category":"分类"}]
```

### 7.3 选择题 Prompt 结构

```
[System] 你是一位专业的技术面试官，擅长出选择题。请严格返回合法的JSON数组。

[User]
{上下文: 简历 / 分类 / 两者}

请根据以上信息生成20道选择题（四选一），要求：
1. 每道题有A、B、C、D四个选项，其中只有一个正确答案
2. 覆盖选中技术领域的各个知识点
3. 难度分布：简单约30%、中等约50%、困难约20%
...

返回纯JSON数组，格式：
[{
  "index":0,
  "question":"题目内容",
  "options":{"A":"选项A","B":"选项B","C":"选项C","D":"选项D"},
  "correctAnswer":"B",
  "explanation":"解析说明",
  "category":"technical"
}]
```

## 8. 容错设计

### 8.1 JSON 解析三级降级

```
AI 响应
  │
  ├─ 第一级: Jackson 标准解析 (JsonUtils.parseJsonList)
  │    └─ 成功 → 返回结果
  │
  ├─ 第二级: 正则提取 (parseChoiceQuestionByRegex)
  │    └─ 提取 question / correctAnswer / explanation 字段
  │    └─ 成功 → 返回结果（options 用默认值填充）
  │
  └─ 第三级: 兜底题目
       └─ 生成 20 道预设 Java 选择题（相同结构）
```

## 10. 已知问题与修复

### 10.1 `String.format` 百分号转义 Bug

**现象**：创建选择题面试时返回 400，日志报 `Conversion = '、'`

**原因**：`AIServiceImpl` 中 `String.format()` 的 prompt 包含 `30%、`，Java 的 `Formatter` 将 `%` 视为格式说明符前缀，尝试将 `、` 解析为转换符失败。

**修复**：在 `String.format` 的模板字符串中将 `%` 转义为 `%%`：

```java
// 修复前
"3. 难度分布：简单约30%、中等约50%、困难约20%\n" +

// 修复后
"3. 难度分布：简单约30%%、中等约50%%、困难约20%%\n" +
```
